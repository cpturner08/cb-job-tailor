# Base Resume Design Contract

The retained DOCX is the visual authority. Preserve it unchanged and create a
working copy for each approved application.

## Page system

- 2 pages, 1 section, US Letter portrait.
- Page size: 12,240 × 15,840 twips.
- Margins: 864 twips (0.6 in) on every side.
- Single column; no header or footer content.

## Typography and color

- Arial throughout; default body size 10 pt.
- Name: 28 pt, bold, uppercase, centered, navy.
- Section labels: uppercase, bold, tracked letters, navy `#1F3864`, with a
  thin navy rule above.
- Role titles: bold black; dates bold gray and right-aligned on the same line.
- Employer/client/location line: gray.
- Links: blue and underlined.

## Flow

1. Centered name and one-line contact row.
2. Summary.
3. Skills.
4. Reverse-chronological experience with real bullets.
5. Technical skills by bold category.
6. Education.

## Fidelity gates

- Preserve page geometry, type hierarchy, colors, rules, bullet indents, and
  the role/date alignment pattern.
- Never shrink text silently to force content into the retained layout.
- Render and inspect every page after editing.
- The current source begins page 2 with the continuation of Desktop Support
  Manager. A tailored version may paginate differently if content changes, but
  it must not orphan a role heading or clip bullets.
